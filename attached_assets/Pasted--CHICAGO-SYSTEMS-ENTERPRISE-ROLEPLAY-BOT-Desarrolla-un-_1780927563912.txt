# CHICAGO SYSTEMS - ENTERPRISE ROLEPLAY BOT

Desarrolla un bot profesional para Discord llamado "Chicago Systems", utilizando Node.js, TypeScript, Discord.js v14, PostgreSQL y Prisma/Drizzle ORM.

El bot debe ser un sistema central para comunidades de roleplay y gestión institucional. No debe incluir MDT, CAD, despacho policial ni sistemas de llamadas 911.

## Objetivo

Crear un proyecto completo, listo para producción, modular, escalable y mantenible para servidores de 5.000 a 20.000 usuarios.

RESTRICCIÓN ABSOLUTA: SIN INTERFAZ WEB

El proyecto NO debe incluir ningún tipo de interfaz web.

NO utilizar:

* React
* Next.js
* Vue
* Angular
* Express Dashboard
* Panel Web
* API pública para usuarios
* Frontend
* Sitio web administrativo
* Dashboard HTML
* Dashboard de estadísticas web
* Login web
* Portal web

Toda la administración debe realizarse exclusivamente desde Discord mediante:

* Slash Commands
* Botones
* Select Menus
* Modales
* Embeds
* Hilos automáticos
* Menús interactivos

Los administradores deben poder configurar el sistema completo desde Discord sin editar archivos manualmente.

Todas las configuraciones deben almacenarse en PostgreSQL y modificarse mediante comandos administrativos.

Ejemplos:

✓ Correcto:

* /config economia
* /config marketplace
* /config impuestos
* /config departamentos
* Paneles con botones
* Select Menus
* Modales

✗ Incorrecto:

* https://dashboard.chicagosystems.com
* Panel administrativo web
* Página de estadísticas
* Configuración mediante navegador
* Login mediante navegador

Chicago Systems debe funcionar como una aplicación 100% nativa de Discord, sin depender de ningún sitio web para operar o administrarse.
